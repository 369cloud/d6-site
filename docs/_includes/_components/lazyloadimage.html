JS 部分

{% highlight js %} domReady(function(require){ $('.ui-lazy-loader').lazyloadimage(); }); {% endhighlight %}

通过lazyloadimage方法返回一个或多个图片懒加载对象的数组（根据选择器选到的dom对象个数）

{% highlight js %} var lli = $('.ui-lazy-loader').lazyloadimage(); //单个 var llis = $('.ui-lazy-loader').lazyloadimage(); //数组 {% endhighlight %}

样式说明

  • ui-lazy-loader - 图片懒加载组件顶层标示。 必选 元素。
  • ui-lazy-loader-content - 图片懒加载组件内容区域样式。 必选 元素。
  • ui-lazy - 延迟加载项样式 必选 元素。
