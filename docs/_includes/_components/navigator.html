样式说明

  • ui-navigator - 顶部导航组件标示。 必选 元素。
  • ui-navigator-list - 顶部导航列表样式 必选 元素。

属性

属性 取值范围 默认值 说明
active Number 0 初始选中下标（从0开始）

事件

在操作过程中可以通过以下事件来控制顶部导航的操作。

事件 参数 说明
switch event, to, li ，其中 to 当前选中的下标， li 选中的html对象 发生切换时触发

方法

在操作过程中可以通过以下方法来控制底部导航的操作。

方法 参数 说明
switchTo index ：目标的下标 切换到目标项
