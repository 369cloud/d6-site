样式说明

  • ui-switch - switch组件标示。 必选 元素。
  • ui-active - switch组件默认开启标示。 可选 元素。
  • ui-switch-handle - switch组件滑动按钮样式 可选 元素。
  • ui-badge - 标记样式。 必选 元素。
  • ui-badge-red - 标记颜色样式（紅色）。 可选 元素。
  • ui-btn - 按钮样式。 必选 元素。
  • ui-btn-blue - 按钮颜色样式。 可选 元素。
